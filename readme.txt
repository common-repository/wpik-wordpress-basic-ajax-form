== Description ==
WordPress Plugin that adds AJAX based data Insertion and Retrieval. By default it is added to WP page content area, but you can modify it based on your needs. You can use this plugin as the base form plugin for creating WP AJAX Forms.

= Minimum Requirements =
* WordPress 5.2 or greater
* PHP version 7.3 or greater (PHP 7.3 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)

= Automatic installation =
1. Log in to your WordPress dashboard
2. Navigate to the Plugins menu and click Add New.
3. Search and locate 'Wpik WordPress Basic Ajax Form'.
4. Click 'Install Now'

= Manual installation =
The manual installation method involves downloading Wpik WordPress Basic Ajax Form' plugin and uploading it to your webserver via FTP application. You may find more 
instructions on how to do this here (http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==
= Can I go back to WordPress default at a later point? =
Yes. you can uninstall the plugin at any stage.
